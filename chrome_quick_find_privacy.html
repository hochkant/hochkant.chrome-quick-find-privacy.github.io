1) Overview

Chrome Quick Find is a browser extension that provides one fast search box to locate and jump to your bookmarks, folders, open tabs, and tab groups. The Extension is designed to process data locally on your device. We do not operate servers for this Extension, do not collect analytics, do not run ads, and do not sell or share personal data.

2) Scope of this policy

This policy covers:

  • The Extension: “Chrome Quick Find” published on the Chrome Web Store.
  • The Project Website (if any): a static site hosted on GitHub Pages that contains documentation and this policy.

3) Data processing in the Extension

What the Extension accesses (locally):

  • Bookmarks (title, URL, folder path) via bookmarks — read-only, for search and “open on click”.
  • Tabs (title, URL, window) via tabs — to list current tabs and activate the selected tab; no page content is read.
  • Tab groups (name, color, collapsed state) via tabGroups — to list/filter/focus groups.
  • Top sites (most-visited list) via topSites — optional local suggestions; can be disabled in settings.
  • Local preferences & cache via storage — user settings (e.g., theme, toggles) and a small local cache to speed search.
  • Favicons — obtained through Chrome’s favicon mechanisms to display icons next to results.

What we do not access or do:

  • No reading of page contents; no broad host permissions.
  • No transmission of personal data to our servers (we run none for the Extension).
  • No analytics, tracking pixels, advertising networks, fingerprinting, or profiling.
  • No sale or sharing of data with third parties.

User controls: You can disable “Top Sites” suggestions, clear Extension data in settings, and uninstall the Extension at any time (see Retention & deletion).

8) Security

Source code is kept under version control and changes are reviewed before release. No method of electronic transmission or storage is fully secure; please use the latest Chrome version and keep your system updated.

9) Retention & deletion

  • Extension data: User preferences and cache are stored in chrome.storage on your device. You can clear them in the Extension settings or by uninstalling the Extension (which removes local data).
  • Website: We do not maintain server-side logs. GitHub Pages may keep logs per its own policies (see GitHub Privacy Statement).

Uninstall instructions: In Chrome, go to chrome://extensions → find “Chrome Quick Find” → Remove.

Appendix — Chrome permissions mapping

  • bookmarks — Read bookmark metadata (title, URL, folder path) to display search results; read-only.
  • tabs — List open tabs and activate a selected tab; page content is not read.
  • tabGroups — Read group name/color/collapsed state to list and focus groups.
  • topSites — Optional local suggestions of most-visited sites; can be disabled.
  • storage — Store user preferences and a small cache locally (or Chrome sync, if enabled by the user/browser).
  • Favicons — Display icons via Chrome’s built-in favicon mechanisms.
